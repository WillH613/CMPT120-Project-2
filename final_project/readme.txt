Periphery Readme
by Will Hammer

Text Commands: 

n, N to go North
s, S to go South
e, E to go East
w, W to go West
i displays inventory
attack is used to attack if you have a weapon

**HINT** Kill Roderick to finish the game
**HINT** If you are at the church, you require a weapon of some sort if you want to kill Roderick
**HINT** The most direct route to finish is as follows: "North, North, North, North, West, North, West, North, West(take ax), east, east, east"
**HINT** Taking items gives you points
